Creating a RESTful API with Express

Setup

To get started let's install some dependencies (assuming you have nodejs, npm, and yarn installed

yarn install --dev express

yarn install --dev cors

yarn install --dev body-parser

Hello World Example

Here is a simple hello world example where calling our API at the base bath "/" returns a JSON object with the message "welcome to my API"

server.js

var express = require('express')
var cors = require('cors')
var app = express()
var bodyParser = require('body-parser')
app.use(bodyParser.json())
app.use(cors())
var port = process.env.PORT || 8080

app.get('/', function (request, response) {
  response.json({
    welcome: 'welcome to my API!'
  })
})
app.listen(port)

Let's run the server now

node server.js

Open http://localhost:8080 in your browser or through postman and you should see the JSON response message

{
  welcome: 'welcome to my API!'
}
